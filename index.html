How to Use This UI (Prototype)

  1. Power On: Click Power. The machine enters Preheating, then switches to Ready with three long beeps.
  2. Select a Mode: Choose Single, Double, Steam, or Clean. The selected mode will highlight.
  3. Start / Stop: Click Start to run the selected mode. While running, the button label changes to Stop. Click Stop to cancel the operation.
  4. Temperature: When the machine is Ready, click Temp to cycle Low → Medium → High.
  5. Low Water (Demo Toggle): Hold Shift and click Power to toggle Low Water. When active, the yellow light blinks and all controls are locked except Power. Clicking locked controls triggers two short beeps.

Design Improvements

  • Thicker horizontal bar buttons improve physical affordance.
  • Mode selection is separated from execution (Start/Stop) to reduce accidental actions.
  • Clearer LED behaviors and consistent feedback improve learnability.
  • Low Water warning prevents operational errors and supports error prevention.
